1.搭建博客所需要的工具

Node.js下载
Git下载（git是开源的分布式控制系统，可有效管理项目）
hexo下载（npm install hexo-cli -g)

检验工具是否安装成功
windows+R键打开窗口输入cmd进入页面

node -v
npm -v
git --version
npm install hexo-cli -g
hexo -v(检查版本号）

2.Github搭建仓库

1.注册github账号并登录。
2. 进入页面搭建仓库（利用github仓库去存储博客内容，注意仓库名应为用户名.github.io) 按图示即可搭建仓库。

  1. 仓库搭建成功图示

    3.生成SSH Keys

    打开任意文件夹，右键打开Git Bash Here，进入页面
    $ ssh
    $ ssh-keygen -t rsa -C “邮箱地址”
    （四次回车后生成文件.ssh，找到id_rsa.pub文件复制文件内容，打开github在个人头像下找到settings点击后进入页面找到SSH and GPG keys，于SSH keys 点击new，将复制内容添加于Key一栏，生成密钥。）
    $ ssh -T git@github.com(测试是否绑定成功）
    命令执行后输入yes继续命令，出现Hi 用户名！即绑定成功。

4.本地生成博客内容

在任意盘内新建文件夹，进入文件夹于空白处右键Git Bash Here生成窗口。

$ npm install -g hexo    #安装hexo
$ hexo init    #初始化hexo（start blogging with hexo出现即初始化成功）
$ hexo s     (静态生成本地hexo页面，复制链接浏览器打开出现以下内容）


5.发布博客到互联网

打开上述文件夹内寻找_config.yml文件打开进行修改。

  deploy：
      type：git
      repo： https://github.com/用户名/仓库名.git
      branch: main


再次在文件夹内打开Git Bash Here

 $ npm install hexo-deployer-git --save
 $ hexo g
 $ hexo d   (该命令进行中会弹出窗口 输入我们的用户名后，弹出输入密码，此时我们打开github，在个人头像下settings进入页面找到Developer settings点击找到personal access tokens点击new建立令牌，赋予权限后生成令牌，将令牌生成的序列号当做密码输入弹窗，当出现Deploy done：git 即成功。）





最后搭建成功！！！通过用户名.github.io即可访问。
